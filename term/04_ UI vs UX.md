# UI vs UX

```jsx
UI와 UX는 모두 사용자 경험에 관련된 요소지만, 약간의 차이가 있어요. 
UI는 사용자가 제품 또는 서비스와 상호 작용할 때 
볼 수 있는 것들을 다루는 것이고, 

UX는 사용자가 제품 또는 서비스를 사용하는 전반적인 경험을 다룹니다. 
간단히 말하면, 

UI는 디자인, 레이아웃, 그래픽 등을 다루고, 
UX는 사용자가 느끼는 전반적인 만족도와 편의성을 고려합니다. 
예를 들어, 어플리케이션의 UI는 버튼, 아이콘, 색상 등을 다루고, 
UX는 그 어플리케이션을 사용하는 동안 사용자가 겪는 흐름, 편의성, 만족도 등을 고려합니다.
```
